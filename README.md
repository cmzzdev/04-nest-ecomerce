# Ecomerce API

1. Clone project

2. Open terminal in folder root and execute: 
```
yarn install
```` 

3. Clone __.env.template__ and rename to __.env__

4. Fill out the environment variables in __.env__

5. Run db
```
docker-compose up -d
```

6. Run App (DEV)
```
yarn start:dev
```

7. Load fake Data
```
http://localhost:3000/api/seed
```
